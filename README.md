# Clove — Landing Page (Test Assignment)

**Live demo: https://codingbee-gif.github.io/clove-landing-assignment/ 


## Project Overview
This repository contains a responsive landing page built from a Figma design using **only HTML5 and CSS3**.  
The project is a static frontend implementation created as part of a test assignment to translate a Figma design into a responsive web page.

## Sections Implemented
The page implements the following sections (6 selected sections from the Figma design):
1. **doctor Section** (Painless root canal — CTA)  
2. **Appointment Form** (right-side form layout)  
3. **Clinics Statistics** (horizontal list)  
4. **Types of Root Canal Treatments** (card layout)  
5. **Testimonials / Real Transformations** (image strip)  
6. **Footer** (branding, links, copyright)

## Features
- Responsive layout for **mobile / tablet / desktop** breakpoints  
- Semantic HTML structure (headings, paragraphs, lists)  
- CSS-only layout using **Flexbox** (and minimal grid)  
- Exported images used from the Figma file and stored in `/images`  
- Simple, clean folder structure ready for deployment
